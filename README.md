# 📊 Seaborn Data Visualization (Learning Project)

## 📌 Overview
This repository showcases my learning journey with **Seaborn**, a powerful Python library for statistical data visualization. It includes practical implementations of various plots used in **Exploratory Data Analysis (EDA)** on real-world datasets.

The goal of this project is to understand data patterns, relationships, and distributions using visual techniques.

---

## 📚 Topics Covered
- Line Plot
- Bar Plot
- Scatter Plot
- Histogram
- Box Plot
- Violin Plot
- Count Plot
- Heatmap
- Pairplot
- Distribution Plot

---
